# Text-Similarity
Text-Similarity can divide into two part:  Representation and Calculate Similarity.

1. Representaiton  
   - Topic model
   - TF-IDF
   - Sentence Vector
   - Word Vector
   - Simhash
2. Similarity  
   - Minimum distance
   - Cosine distance
   - Euclidean distance
   - Hamming distance
   - Classifier
